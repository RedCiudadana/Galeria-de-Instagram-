Overview

Simple Instagram Fancybox is easy to use and has an extensive selection of options. We listened to your feature requests and created a brand new product that includes them all. See the list below for more details!

Key Features

  • User Instagram Feed Mode - This feed mode allows you to specify a user id to pull in photos from a specific user.
  • Fancybox 2 Integration Creates a stylish popup shadowbox effect for your gallery.
  • Fully Customizable - Features an array of options to allow you to customize your experience to fit your needs.

Usage

1. First, make sure you are using valid DOCTYPE.

2. Include necessary JS files.

<script type="text/javascript" src="../jquery-1.11.0.min.js"></script>
<script type="text/javascript" src="../fancybox2/source/jquery.fancybox.pack.js"></script>
<script type="text/javascript" src="../simpleInstagramFancybox.js"></script>
<script type="text/javascript" src="js/demo.js"></script>

3. Include necessary CSS files.

<link href="../fancybox2/source/jquery.fancybox.css" rel="stylesheet" >
<link href="css/demo.css" rel="stylesheet">
<link href="../simpleInstagramFancybox.css" rel="stylesheet">

4. Create the HTML container you wish to display your gallery in.

	<div class="element"></div>

5. Activate the plugin.

$('.element').simpleInstagramFancybox({
   mode : 'user',
   userID : 'YOUR_NUMERIC_USERID',
   accessToken : 'YOUR_ACCESS_TOKEN',
   speed : 700,
   delayInterval : 80,
   appearEffect : 'fade',
   captionOn : false
});

API & Options

Now that you've installed the plugin and explored the different options, it is time to make some decisions and configure your element. Take a look at the list below to see the available shortcode options.

  • accessToken - String: This a mandatory setting that allows you to specify a user token.
  • mode - String: (user) Sets the Instagram feed mode.
  • userID - String: This is a setting that you have to use if your using "user" mode".
  • speed - Integer: Sets the speed of the images fade in effect. Default is 700.
  • delayInterval - Integer: Sets the interval of the delay between photos appearing. Default is 80.
  • appearEffect - Allows you to set the effect used to show photos. Options include fade,slide,motion. Default is fade.
  • captionOn - This setting allows you to turn on captions. If this is set to true, it will automatically pull captions in from the instagram feed. Options include true or false.
